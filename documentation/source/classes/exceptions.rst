Resource Error
~~~~~~~~~~~~~~

.. cpp:class:: FILE_OPEN_FAILED

	Could not open file.

.. cpp:class:: READING_FILE_ERROR

	Error while reading the file.

.. cpp:class:: TIME_BADLY_FORMATED

	The time we are trying to read is badly formatted.

.. cpp:class:: ANGLE_BADLY_FORMATED

	The angle we are trying to read is badly formatted.

.. cpp:class:: TIMETABLE_BADLY_FORMATED

	The timetable is badly formatted

Network Error
~~~~~~~~~~~~~

.. cpp:class:: STATION_NOT_IN_SCHEDULE

	The station is not in the Timetable

	No Timetable was found for the day

.. cpp:class:: CANNOT_ORDER_SAME_STATION

	The station Being compared order are the same.

.. cpp:class:: STATION_NOT_IN_NETWORK

	The station we are trying to find is not part of the network.

.. cpp:class:: STATION_NOT_IN_LIST

	The station we are trying to find is not part of the list.
